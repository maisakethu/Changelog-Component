# Responsive Changelog Component

A clean and responsive changelog component built with semantic HTML and modern CSS. This component is designed for easy integration into a website without JavaScript.

## Files

- `index.html` — markup for the changelog component
- `styles.css` — styling for card layout, timeline, responsiveness, and hover effects

## Features

- Semantic HTML structure using `main`, `section`, `article`, and `time`
- Card-style layout with centered page design
- Vertical timeline feel with entry markers
- Responsive design for desktop, tablet, and mobile
- Hover animations for entry highlight
- Google Fonts integration for polished typography
- No JavaScript required

## Usage

1. Open `index.html` in your browser.
2. Customize entries inside the `.timeline` container.
3. Adjust colors, spacing, and fonts in `styles.css` as needed.

## Customization

- Add or remove `.timeline-entry` items in `index.html`
- Update `entry-meta` details for date and version
- Modify the CSS variables at the top of `styles.css` for theme changes

## Notes

- The layout is mobile-first and stacks content vertically on small screens.
- The component uses Flexbox/Grid for layout and modern visual styling.
